# In-Memory Python Console Todo Application

A command-line todo application demonstrating spec-driven development principles.

## Overview

This project is a minimal viable product for task management via command-line interface, suitable for hackathon projects or learning spec-driven development.

## Features

- Add tasks with title and description
- View task list with status indicators
- Update task details
- Delete tasks by ID
- Mark tasks as complete/incomplete
- All data stored in memory (session-scoped)
